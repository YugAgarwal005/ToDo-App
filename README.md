# ToDo-App

A simple and efficient web-based task management application to help you organize your daily tasks.

## Live Link

🔗 **[Visit the Live App](https://yugagarwal005.github.io/ToDo-App/)**

## Features

- ✅ Create new tasks
- ✏️ Edit existing tasks
- 🗑️ Delete tasks
- ✔️ Mark tasks as complete/incomplete
- 📱 Responsive design
- 🎨 Clean and intuitive user interface

## Project Structure

```
.
├── index.html          # Main HTML file
├── script.js           # JavaScript functionality
├── style.css           # Styling
├── README.md           # This file
├── music-tone/         # Audio files directory
└── svg/                # SVG assets directory
```

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)

## How to Use

1. Visit the [live app](https://yugagarwal005.github.io/ToDo-App/)
2. Enter a task in the input field
3. Click the add button or press Enter to add the task
4. Click the checkmark icon to mark a task as complete
5. Click the delete icon to remove a task
6. Use the edit option to modify existing tasks

## Getting Started Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/yugagarwal005/ToDo-App.git
   ```

2. Navigate to the project directory:
   ```bash
   cd ToDo-App
   ```

3. Open `index.html` in your web browser
